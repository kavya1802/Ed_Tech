## 📌 Project Overview

This end-to-end data science project analyzes campus placement and hiring data
to identify key factors that determine whether a student gets placed or not,
and builds machine learning models to predict placement outcomes.

> **45% of students leave campus without a job offer despite academic qualifications.
> What really determines who gets placed?**

---

## 📁 Dataset

| File | Records | Description |
|------|---------|-------------|
| Student_Profile.xlsx | 15,215 | Age, Gender, CGPA, Degree, Region, Attendance, Internship |
| Academic_Skills.xlsx | 15,215 | Technical Score, Quantitative Score, Soft Skills, Projects, Hackathon |
| Placement_Record.xlsx | 15,215 | Placement Status, CTC, Company, Industry, Placement Type |

**After merging & cleaning → 13,791 rows × 27 columns**
